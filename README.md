# Pokémon Search - FM Tech

A Pokémon search web application built with **Next.js**, **TypeScript**, **GraphQL**, and **Apollo Client**.  
Users can search for Pokémon by name and view detailed information including stats, attacks, and evolutions.

---

## Live Demo

- https://search-pokemon-fm-tech-two.vercel.app/  

---

## Features

- Search Pokémon by name
- Display Pokémon details
- Empty state when no search keyword is provided
- Not found state when Pokémon does not exist
- Loading indicator while fetching data
